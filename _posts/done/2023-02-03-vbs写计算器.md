# 基本（加减乘除）
## 内容
```
msgbox"欢迎使用VBS计算器",,"欢迎使用"
dim a,b,c,m
a=inputbox("请输入第一个数字")
c=inputbox("请输入计算符号")
b=inputbox("请输入第二个数字")
if c="+" then m=CDbl(a)+CDbl(b)
if c="-"then m=a-b
if c="*"then m=a*b
if c="/"then m=a/b
msgbox m,,"计算完毕"
```
