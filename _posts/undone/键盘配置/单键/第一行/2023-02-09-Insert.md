# 1 原位置

`Insert`

# 2 配置方式

没有映射，Obsidian 也不支持 单键，但是如果映射它好像会影响 Quicker 的运行

# 3 +Shift

`Shift` + `Insert` 是默认粘贴

# 4 +Ctrl

`Regex Pipeline:Apply Ruleset` 快捷键设置为 `Ctrl` + `Insert`

# 5 可能性 

`Ctrl` + `Shift` + `Insert` 因为离得太远，所以用 `Ctrl(Left)` + `Esc` 映射了，现在是 `截图并自动复制` 的快捷键
