# 1 原位置

`F1`

# 2 配置方式

不映射，直接使用 `F1`

# 3 Obsidian 配置

## 3.1 单键

`插入有序列表` 设置为 `F1`

### 3.1.1 理由

html 里有序列表要 Tab 才能显示。这样插入有序列表离 Tab 键近

# 4 +Shift

`打开快速切换` 设置为 `Shift` + `F1`

## 4.1 原因

打开快速切换后主要要搜索笔记，用的是键盘输入区，所以可以把对应的快捷键放在左边
